## Способ 2: Собрать EXE самостоятельно (если EXE нет)

### Шаг 1. Установите Python (один раз, навсегда)

1. Откройте https://www.python.org/downloads/
2. Скачайте последнюю версию Python 3.12+
3. Запустите установщик
4. **ВАЖНО:** на первом экране поставьте галочку **"Add Python to PATH"**
5. Нажмите **Install Now**
6. После установки — закройте окно

### Шаг 2. Соберите программу

1. Распакуйте архив с программой в любую папку (например `C:\Procurement\`)
2. В этой папке зажмите **Shift** и кликните правой кнопкой по пустому месту
3. Выберите **"Открыть в терминале"** (или **"Открыть окно PowerShell здесь"**)
4. В окне терминала введите:
   ```
   python build.py
   ```
5. Нажмите Enter и подождите 1-2 минуты
6. По завершении в папке появится подпапка `dist\` — внутри файл **Procurement.exe**

### Шаг 3. Запуск

Двойной клик по `Procurement.exe` — программа запустится.

Можете перенести этот один файл куда угодно (на флешку, на другой компьютер с Windows) — он работает автономно.

## Как пользоваться программой

1. **Шаг 1** — выберите страну (Узбекистан и др.) и регион
2. **Шаг 2** — добавьте позиции ТМЦ:
   - Кнопка **"+ Добавить"** — заполняете название, параметры, кол-во
   - Или загрузите готовый Excel-файл (формат: Наименование, Параметры, Ед.изм., Объём)
3. Нажмите **"🔍 Найти поставщиков"**
4. Программа сама сходит на маркетплейсы (OLX.uz, Glotr.uz и т.д.) и найдёт реальных поставщиков с актуальными ценами
5. **Шаг 3** — посмотрите результаты:
   - По 3 поставщика на каждую позицию (дешёвый/средний/дорогой)
   - Кликайте на ссылки чтобы открыть карточку товара
6. Кнопка **"⬇ Экспорт в Excel"** — сохраняет красивый отчёт

## Если возникли проблемы

| Проблема | Решение |
|---|---|
| `python: команда не найдена` | Не установили Python или забыли галочку "Add to PATH" — переустановите |
| `pip install` падает | Запустите PowerShell от имени администратора |
| .exe не запускается, выдаёт ошибку про DLL | Установите Microsoft Visual C++ Redistributable (бесплатно с сайта Microsoft) |
| Поиск не находит поставщиков | Проверьте интернет; некоторые сайты могут временно блокировать запросы |
| Антивирус ругается на .exe | Это ложное срабатывание (PyInstaller). Добавьте файл в исключения |

## Системные требования

- Windows 10 или 11 (64-bit)
- ~50 MB свободного места
- Интернет (только для поиска поставщиков)
- Microsoft Edge WebView2 (обычно уже установлен в Windows 10/11)

Если WebView2 не установлен — программа покажет ссылку на скачивание (бесплатно у Microsoft).
