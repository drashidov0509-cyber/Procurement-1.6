# 📋 Загрузка файлов через браузер (без скрипта)

## Почему скрипт не работает
Антивирус или Windows Defender блокирует Python от подключения к GitHub.
Решение — сделать то же самое через браузер (он не блокируется).

---

## ШАГ 1. Войдите в GitHub

1. Откройте **https://github.com/login**
2. Введите email и пароль от вашего аккаунта `dilshod0509-cyber`
3. Нажмите **Sign in**

---

## ШАГ 2. Создайте репозиторий

1. Нажмите **«+»** (плюс) в правом верхнем углу → **«New repository»**
2. Заполните:
   - **Repository name:** `procurement-global`
   - **Public** (оставьте как есть)
   - Поставьте галочку ✅ **«Add a README file»**
3. Нажмите зелёную **«Create repository»**

---

## ШАГ 3. Загрузите файлы

1. В созданном репозитории нажмите **«Add file»** → **«Upload files»**

2. Откройте папку `desktop_app` на компьютере

3. Выделите ВСЕ файлы (Ctrl+A) и перетащите в браузер:
   - app.py
   - api.py
   - scrapers.py
   - classifier.py
   - excel_export.py
   - ui.html
   - requirements.txt

4. Внизу нажмите **«Commit changes»**

---

## ШАГ 4. Загрузите папку .github (ВАЖНО!)

Это самый важный файл — без него сборка не запустится.

1. Снова **«Add file»** → **«Upload files»**
2. Перетащите ТОЛЬКО файл:
   `.github/workflows/build.yml`
   
   ⚠️ ПРОБЛЕМА: браузер не принимает файлы из скрытых папок (`.github`)
   
   **Решение:**
   - Откройте файл `.github/workflows/build.yml` в Блокноте
   - Скопируйте всё содержимое (Ctrl+A → Ctrl+C)
   - В репозитории нажмите **«Add file»** → **«Create new file»**
   - В поле **«Name your file»** введите точно: `.github/workflows/build.yml`
   - В большое поле ниже вставьте скопированное содержимое (Ctrl+V)
   - Нажмите **«Commit new file»**

---

## ШАГ 5. Проверьте запуск сборки

1. В репозитории откройте вкладку **«Actions»** (сверху)
2. Должна появиться сборка **«Build Windows EXE»** с жёлтым кружком 🟡
3. Подождите 5-7 минут пока не появится зелёная галочка ✅

---

## ШАГ 6. Скачайте готовый .exe

1. Кликните на завершённую сборку (зелёная галочка)
2. Прокрутите вниз до раздела **«Artifacts»**
3. Нажмите **«Procurement-Windows»** — скачается ZIP
4. Распакуйте ZIP → внутри ваш **Procurement.exe**

🎉 Готово!
