# 🚀 Получить Procurement.exe — ОДИН ШАГ

## Что нужно сделать

1. Распакуйте архив `procurement-desktop.zip` в любую папку
   (например: `C:\Procurement\`)

2. Откройте эту папку в проводнике

3. Зажмите **Shift** + правый клик мыши на пустом месте папки

4. Выберите **«Открыть в терминале»**
   (или «Открыть окно PowerShell здесь»)

5. Введите команду и нажмите **Enter**:
   ```
   python setup_github.py
   ```

6. Ждите ~7 минут — скрипт сам всё сделает и положит `Procurement.exe` в эту же папку

---

## Если Python не установлен

Скачайте с официального сайта:
👉 https://www.python.org/downloads/

При установке обязательно поставьте галочку:
✅ **Add Python to PATH**

После установки повторите шаг 5.

---

## Что делает скрипт

1. ✅ Проверяет токен GitHub
2. ✅ Создаёт репозиторий `procurement-global` в вашем аккаунте
3. ✅ Загружает все файлы программы
4. ✅ Запускает автоматическую сборку на серверах GitHub
5. ✅ Ждёт (~5 мин) и скачивает готовый `Procurement.exe`
6. ✅ Напоминает удалить токен

## После получения .exe

⚠️ **Удалите токен** (он больше не нужен):
👉 https://github.com/settings/tokens
→ Нажмите **«Delete»** рядом с токеном `procurement-build`
